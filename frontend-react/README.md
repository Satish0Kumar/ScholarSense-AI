# ScholarSense React Frontend

React + Vite + Tailwind CSS frontend for ScholarSense AI Academic Intelligence System.

## Setup

```bash
# Install dependencies (already done)
npm install

# Start development server
npm run dev
```

## Backend Connection

- Backend API: `http://localhost:5001/api`
- Ensure Flask backend is running on port 5001

## Default Accounts

- **Admin**: admin@scholarsense.com / admin123
- **Teacher**: teacher@scholarsense.com / teacher123

## Structure

- **Dashboard** - Overview metrics
- **Students** - Unified list + inline profile view
- **Academics** - Attendance | Marks | Behaviour tabs
- **Risk & Analytics** - Predictions | Batch | Analytics tabs
- **Alerts** - Notifications | Parent Portal tabs
- **User Management** - Admin only

## Tech Stack

- React 19
- Vite 8
- React Router 7
- Axios
- Tailwind CSS 4
